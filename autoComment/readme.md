# 🚀 프로젝트명: YouTube Personal Agent

## 1. 프로젝트 목적 및 주요 기능
### 1.1 개발 목적
* **Gemini Pro**를 활용하여 타겟 채널의 멀티미디어 콘텐츠(영상, 쇼츠, 게시글)를 깊이 있게 이해하고, 사용자의 정체성이 담긴 고품질 댓글을 자동 생성.
* 단순 반복을 넘어, 언어 장벽을 허물고(다국어 지원) 인간적인 터치(사용자 승인 및 가변 스케줄링)를 가미한 에이전트 구축.

### 1.2 주요 기능
* **콘텐츠 심층 분석:** Gemini Pro의 멀티모달 능력을 활용해 영상 제목, 설명, 자막 등을 파악하여 맥락에 맞는 댓글 생성.
* **콜드 스타트(Cold Start) 지원:** 기존 댓글 이력이 없는 경우, 5가지 기본 스타일 페르소나를 제안하여 선택 가능하게 함.
* **글로벌 대응 (Bilingual):** 외국어 콘텐츠를 한글로 번역/이해한 뒤, [한글 + 해당 언어] 병기 댓글 생성.
* **정밀 스케줄링:** 3시간 단위로 작동하되, 이전 실행 시점의 '분(Minute)' 단위를 피하고 '초(Second)' 단위에도 랜덤 지연(Jitter)을 주는 로직 적용.

---

## 2. 기술 스택 및 개발 환경
* **LLM Engine:** Google Gemini Pro (`google-generativeai`)
* **언어:** Python 3.x
* **인증 및 API:** * YouTube Data API v3 (Google OAuth 2.0 기반 인증)
    * `google-auth-oauthlib` (Refresh Token을 통한 세션 유지)
* **웹 자동화:** `selenium` 또는 `playwright` (커뮤니티 게시글 스캔 및 쿠키 기반 세션 유지용)
* **메시징:** `python-telegram-bot` (사용자 컨펌 및 알림)
* **데이터 관리:** `SQLite` (콘텐츠 로그, 스케줄링 상태 및 스타일 관리)

## 3. 구현 로직 (Implementation Logic)

### 3.1 스타일 설정 및 학습 (Style Setup)
* **기존 이력 존재 시:** 최근 댓글 데이터를 Gemini에게 전달하여 말투(Few-shot prompting) 학습.
* **이력 미존재 시:** 5가지 예시 스타일(응원형, 질문형, 유머형, 전문가형, 간결형) 중 선택.

### 3.2 콘텐츠 분석 및 댓글 생성 (Analysis & Generation)
1. **내용 파악:** 타겟 콘텐츠의 텍스트 및 메타데이터 수집 및 번역.
2. **프롬프트 엔지니어링:** 사용자의 페르소나와 다국어 병기 규칙을 적용한 초안 생성.

### 3.3 사용자 컨펌 프로세스 (Human-in-the-Loop - 상세)
텔레그램 봇을 통해 생성된 초안을 검토하고 게시 여부를 결정합니다.
1. **알림 전송:** Gemini가 생성한 [댓글 초안]과 [콘텐츠 정보]를 버튼과 함께 전송.
2. **사용자 선택 로직:**
    * **[✅ 그대로 게시]:** 초안이 마음에 들 경우 버튼 클릭 즉시 유튜브 API를 통해 댓글 게시.
    * **[✍️ 직접 수정/거절]:** 초안이 마음에 들지 않을 경우 클릭. 봇이 "수정할 내용을 입력해주세요"라고 응답하며 **입력 대기 상태(Conversation State)**로 진입. 사용자가 텔레그램 채팅창에 원하는 댓글을 직접 입력하여 전송하면, 해당 내용으로 유튜브 게시.
    * **[⏩ 이번 건 건너뛰기]:** 게시를 원치 않을 경우 클릭. 해당 영상 ID를 '스킵' 상태로 기록하고 종료.
3. **피드백:** 게시 완료 후 처리 결과(성공/실패)를 다시 사용자에게 알림.

### 3.4 데이터베이스 스키마 (SQLite)
* **`Content_Log`:** 콘텐츠 고유 ID, 유형, 제목, 생성된 댓글, 최종 게시 댓글, 상태(게시/수정게시/스킵), 타임스탬프.
* **`Bot_Settings`:** 마지막 실행 분(Minute), 활성화된 스타일, OAuth 토큰.

## 4. 상세 스케줄링 및 운영 규칙
* **실행 주기:** 3시간 단위 체크(한국 시간 기준 새벽 1시부터 6시까지는 제외).
* **랜덤 로직:** 이전 실행 분(T)과 겹치지 않는 1~59 사이의 랜덤 `Next_Minute` 설정.
* **알림 응답 대기:** 텔레그램 발송 후 1시간 내 응답이 없으면 '미승인' 처리 후 다음 주기에 재검토.
* **에티켓 시간:** 설정된 심야 시간대(한국 시간 기준 새벽 1시부터 6시까지)에는 알림 발송 중단.

---

## 5. 예외 처리 및 보안 (Anti-Bot & Security)
* **OAuth 보안:** `token.json`을 통한 안전한 권한 관리.
* **봇 감지 회피:** 실제 브라우저 `User-Agent` 및 쿠키 세션 동기화, 초 단위 랜덤 지연.
* **입력 예외:** 직접 수정 모드에서 너무 긴 텍스트나 금칙어가 포함될 경우 경고 메시지 출력.
* **할당량 관리:** YouTube API 할당량 소진 시 즉시 중단 및 알림.
