# Upbit Auto Trading Bot

이 프로젝트는 Upbit API를 활용하여 가상자산(암호화폐)을 자동으로 매매하는 파이썬 기반 프로그램입니다.

## 핵심 기능 (Features)
- **다중 코인 모니터링**: 5개 이상의 코인을 동시에 모니터링하여 병렬적 기회 포착.
- **최적 알고리즘 적용 (변동성 돌파 + 상승장 필터)**:
  - 전일 변동폭(고가-저가)을 당일 시가에 더한 값을 목표가로 계산.
  - 5일 이동평균선(MA5) 이상에서만 매수하도록 필터링하여 상승장에서만 공격적으로 진입.
- **모의 실험 및 실전 매매 분리 (Simulation / Real Mode)**: 환경변수 하나로 리스크 없이 가동 점검 및 백테스팅(모의 운영) 가능.

## 시뮬레이션 모드 테스트 로직 구동 (`test_simulate.py`)
이 모듈은 `main.py`의 함수를 재사용하여, 강제로 가격 변동을 시뮬레이션하고 매수/매도 로직이 정상 동작하는지 짧은 시간(5분) 동안 검증하는 목적의 테스트 스크립트입니다. 
빠른 매매 검증을 원할 경우 `python test_simulate.py`를 실행하세요.

## 🚀 시작하기 가이드 (Getting Started)

### 1단계: Upbit API 키 발급 및 설정
실제 매매를 진행하기 위해서는 Upbit Open API 키가 필요합니다.
1. **Upbit 로그인**: [업비트 마이페이지 - Open API 관리](https://upbit.com/mypage/open_api_management)로 이동.
2. **API 발급**: "Open API 발급 받기" 진행 (카카오/네이버 인증 필요).
3. **권한 설정**: `[자산조회]`, `[주문조회]`, `[주문하기]` 권한을 반드시 체크하세요.
4. **IP 등록**: API 키 도용 방지를 위해 코드가 실행되는 환경의 **고정 IP**를 필수로 등록할 것을 권장합니다.

### 2단계: 패키지 설치
이 프로젝트는 파이썬 전용 업비트 라이브러리인 `pyupbit`와 환경변수 로드를 위해 `python-dotenv`를 사용합니다.
```bash
pip install pyupbit python-dotenv
```

### 3단계: `.env` 환경 변수 설정
프로젝트 루트 경로(`tradeProject/.env`)에 아래와 같이 `.env` 파일을 생성하거나 환경 변수에 키를 세팅하세요.

```env
# Upbit API Keys
UPBIT_ACCESS_KEY="자신의_ACCESS_KEY_입력"
UPBIT_SECRET_KEY="자신의_SECRET_KEY_입력"

# 실행 모드: True면 모의 투자 모드(돈 안 듦), False면 실제 매매
SIMULATION_MODE=True
```

### 4단계: 봇 실행
```bash
python main.py
```
- `SIMULATION_MODE=True` 상태에서는 실제 API 주문이 나가지 않고, 봇 내부에 100만 원(가상)이 있다고 가정하여 주문 결과 및 잔고 변화를 로그로만 출력합니다.
- 동작이 완전히 검증되면 `.env`의 값을 `SIMULATION_MODE=False`로 변경하고 실전 투자를 시작하세요.

## 전략 로직 (Trading Strategy Algorithm)
현재 채택된 `변동성 돌파 + 추세(MA5) 전략`은 다음과 같이 동작합니다.
1. **타겟 가격 수립**: `오늘 시가 + (어제 고가 - 어제 저가) * 0.5`
2. **매수 조건**: 현재가가 '타겟 가격'을 돌파 **동시에** '5일 이동평균선(MA5)' 위에 있을 경우 (상승장일 때만 진입).
3. **매도 조건**: +3% 이익 실현(Take-Profit) 혹은 -2% 손절(Stop-Loss).
